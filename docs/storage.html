Storage

localStorage helpers for persisting connection state across page loads

Overview

The storage module provides typed localStorage accessors for the values the provider uses to restore state on page load. You generally do not call these directly — MeroProvider reads and writes them automatically. They are useful when you need to pre-populate state or implement custom connection UI.

import {
  getNodeUrl, setNodeUrl, clearNodeUrl,
  getApplicationId, setApplicationId, clearApplicationId,
  getContextId, setContextId, clearContextId,
  getContextIdentity, setContextIdentity, clearContextIdentity,
  clearAllStorage,
  localStorageTokenStorage,
} from '@calimero-network/mero-react';

String Key Helpers

Each value has a get, set, and clear function. All values are stored as plain strings.

getNodeUrl() → string | null
Read the persisted node URL. Returns null if not set.
setNodeUrl(url)
Persist the node URL.
clearNodeUrl()
Remove the node URL from storage.
getApplicationId() → string | null
Read the persisted application ID.
setApplicationId(id)
Persist the application ID.
clearApplicationId()
Remove the application ID.
getContextId() → string | null
Read the persisted context ID.
setContextId(id)
Persist the context ID. Provider reads this on mount.
clearContextId()
Remove the context ID.
getContextIdentity() → string | null
Read the persisted context identity (executor public key).
setContextIdentity(id)
Persist the context identity.
clearContextIdentity()
Remove the context identity.
clearAllStorage()
Remove all mero-react keys from localStorage — equivalent to logout cleanup.

localStorageTokenStorage

An async token storage adapter compatible with the TokenStore interface from mero-js.

const localStorageTokenStorage: {
  get(): Promise<TokenData | null>;
  set(token: TokenData): Promise<void>;
  clear(): Promise<void>;
}

This is what MeroProvider uses internally. You can use it directly if you need to read or clear tokens outside of a React component.

Logout Pattern

import { useMero } from '@calimero-network/mero-react';

function LogoutButton() {
  const { logout } = useMero();

  // logout() clears tokens and storage keys automatically
  return <button onClick={logout}>Log out</button>;
}
